# 关于自建图床那些事──typora+cheverto 图床

偶然间发现 typora 可以使用规则，将本地或者网络图片上传到图床。查了些资料，发现了开源的图床 chevereto。正好手上有个闲置的服务器，于是开始了快乐的踩坑之旅

# 4. 页面配置

正常情况下，打开之前映射的网址，能看到数据库配置页面，将之前设置好的数据库和密码输入，同时下一步配置管理员信息

# 5. API 配置
